Установка и удаление пакетов в Xubuntu (1 часть).
Опубликовано 4 июля 2019.

В этой записи мы узнаем, как устанавливать и удалять пакеты в Xubuntu. Это первая часть данной записи. Данная запись должна заменить все видео типа установок всяких приложений в Xubuntu.

Сначала вы можете использовать магазин приложений, встроенный в Xubuntu. С него можно запросто устанавливать приложения. Нужно найти приложение, его установить либо удалить. Но мы будем переходить к более продвинутым способам.

Первый способ — установка пакетов с репозитория Ubuntu. Это делается с помощью команд в терминале. Для поиска приложения в репозиториях, вы можете использовать sudo apt search <название_пакета>. Для установки приложения из репозитория Ubuntu, вы можете использовать команду sudo apt install <название_пакета>. Для удаления приложения, которое было установлено с файла пакета Debian или установленного с репозитория Ubuntu используется команда sudo apt remove <название_пакета>.

Вторым способом является установка приложения с репозитория Snapcraft. Чтобы найти приложение в репозитории Snapcraft, используется команда sudo snap search <название_приложения>. Для установки, соответственно, используется команда sudo snap install <название_приложения>, а для удаления sudo snap remove <название_приложения>.

Третий способ — установка приложения с файла пакета Debian. Для этого нам нужно открыть терминал с той директории, где находится файл пакета, и для его установки используется команда sudo dpkg -i <название_пакета>*. Звёздочка в конце означает обработку всех пакетов, которые находятся в папке после названия, чтобы не писать дополнительно номера версий и тому подобное. Для удаления таких пакетов используется команда sudo apt remove <название_пакета>, о чём упоминалось выше.
