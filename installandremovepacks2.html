Установка и удаление пакетов в Xubuntu (2 часть).
Опубликовано 8 июля 2019.

В этой записи мы узнаем, как устанавливать и удалять пакеты в Xubuntu. Это первая часть данной записи.

Здесь мы узнаем, как устанавливать программы с файла, которое заканчивается на .bundle. Перед установкой мы установим заголовочные файлы ядра с помощью команды sudo apt install build-essential linux-headers-$(uname -r).

Дальше мы должны для того, чтобы выполнять наш bundle-файл, мы должны написать команду sudo chmod +x <название_файла>.bundle, а затем мы откроем наш файл командой sudo ./<название_файла>.bundle. Устанавливаем нашу программу.

Если мы хотим удалить нашу программу, установленную через файл bundle, то предлагаю вам найти инструкцию по удалению вашей программы, поскольку они могут удаляться по-своему.
